# latin_square_solver.py
Python implementation of a Latin Square solver using backtracking.

---

## Features
- Backtracking algorithm
- Solution validation
- Efficient pruning

---

## How to run
```bash
python latin_square_solver.py
```

## Example

Input:
```
N = 3
```

Output:
```
1 2 3
2 3 1
3 1 2
```

---

## About

This project demonstrates recursive backtracking to solve Latin Square problems efficiently.
